- `BRF.png` - Foto program BRF

- `PPLG.png` - Foto program PPLG

## Penggunaan

File-file ini digunakan di:
- Landing page (section Fasilitas)
- Blog section
- Komponen lainnya yang memerlukan foto sekolah

## Catatan

- Pastikan semua file memiliki nama yang sesuai dengan referensi di kode
- Format yang disarankan: PNG dengan ukuran yang dioptimalkan
- File dapat diakses melalui URL: `/img/filename.png`
